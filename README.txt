LogoClient
==========

LogoClient is a standalone Swing application and client library for
communicating with label printers over TCP — sending label layout and data
files, querying status, and transferring files for test and development
purposes.

Part of the Commander4j project — https://www.commander4j.com
Help: http://wiki.commander4j.com
